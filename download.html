There are two ways to download Siesta. You can either download the bundle which includes most components or pick and choose the components that you'd like to use.

Siesta currently offers four components:

  • Core - object mapping
  • HTTP - an interface to jQuery/jQuery-like ajax libraries such as Zepto.
  • Storage - an interface to PouchDB for persistence
  • Performance Monitoring: Logs timing information around mapping, HTTP requests etc.

Bundle

The Siesta bundle includes Core, HTTP and Storage.
